======================
DeepL Translate (CORE)
======================

:Extension key:
    deepltranslate_core

:Package name:
    web-vision/deepltranslate-core

:Version:
    |release|

:Language:
    en

----

This extension provides translation of pages, content and records in TYPO3
for languages supported by `DeepL <https://www.deepl.com/de/docs-api/>`_.
