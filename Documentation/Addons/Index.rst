..  _officialAddons:

Official Addons
===============

Following official addons are available:

..  card-grid::
    :columns: 1
    :columns-md: 12
    :gap: 4
    :class: pb-4
    :card-height: 100

    ..  card:: :guilabel:`PUBLIC` `EXT:deepltranslate_glossary`

        Provides TYPO3 based Glossary management for DeepL Glossaries to use
        for page and content translations.

        ..  rst-class:: horizbuttons-attention-m

        -  `TYPO3 Extension Repository <https://extensions.typo3.org/extension/deepltranslate_glossary>`_
        -  `GitHub <https://github.com/web-vision/deepltranslate-glossary>`_
        -  `Packagist <https://packagist.org/packages/web-vision/deepltranslate-glossary>`_

    ..  card:: :guilabel:`PRIVATE` `EXT:deepltranslate_assets`

        Extend the ability to translate file meta data (FAL) directly as long
        as site-configurations are not ambiguous.

    ..  card:: :guilabel:`PRIVATE` `EXT:deepltranslate_auto_renew`

        Providing auto and renew translation.

    ..  card:: :guilabel:`PRIVATE` `EXT:deepltranslate_mass`

        Provides bulk translation of pages and content.
