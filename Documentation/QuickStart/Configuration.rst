Extension Configuration - API Key
=================================

This extension uses the `DeepL <https://www.deepl.com/de/docs-api/>`_ under
the hood to provide the translations, which requires a API key regarding the
plan and connected limits. Get `your key here` and enter it to the
:guilabel:`extension configuration` for :guilabel:`deepltranslate_core`.

..  figure:: /Images/Reference/configuration-settings-deeplapikey.png
    :alt: :guilabel:`Extension Configuration > Settings > DeepL API Key`
